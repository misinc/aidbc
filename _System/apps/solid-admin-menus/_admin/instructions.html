Description

The admin menu app allows an admin user to change the contents of the admin menus. This app allows you to perform the following features:
  • Change admin menu labels.
  • Add links to external resources.
  • Embed external apps and websites into the admin
  • Reorder and regroup menu items via drag and drop interface.
  • Hide default menu items.
  • Change menu item icons by choosing a different icon from the library or uploading your own.

Recommendations

  • If adding a link that displays within the admin console it is recommended to use the https protocol for your links. This will maximize compatibility of your link. Http links will only work if a user logs into the admin console using http but will not work if logged in using https. Https links will work in both cases.
  • Custom icons need to be PNG files ideally at 20px x 20px with a transparent background. If an image with different dimensions are uploaded the image will be cropped to 20px x 20 px.
  • Keep menu labels under 20 characters otherwise the label will be cut off and after the 19th character there will be three dots (...).

Basic Usage

  • Click on any left menu item's icon to change the icon. You may choose an existing icon or upload a new 20px x 20px PNG file. (You must save your changes before this will take effect.)
  • You may reorder admin menu items by clicking and dragging any admin menu item to a new location within their current scope. Top menu items, left menu items and sub menu items can be individually reordered but not moved to another menu item category.

General Notes

  • Default menu items cannot be removed, although they can be hidden.
  • You can always access the app by going to the URL directly: http://YOURDOMAIN/Admin/AppLoader.aspx?client_id=solid-admin-menus
  • Custom icons need to be PNG Files.
  • You cannot delete default icons from the library.
  • The list under access rights displays the available roles on the website.
  • At least one role needs to have access to the app at all times.
  • Disabling a role from the app removes the "Admin Menu Labels" from the menu. Technically the user can still access the app if they enter the link directly into their browser (limitation in BC API prevent us from completely disabling them).
  • You cannot change the name of this app
  • Top menu items in Business Catalyst do not have icons.
  • You cannot modify the links in the top right.
  • If the user has "Home Page" set as the landing page under "My Details" (this is the default setting), when the user logs in they will be taken to the Dashboard and NOT the first parent item in the menu.
  • To change the roles available under access rights, you must update the roles under Site Settings > User Roles (these are the default labels).
  • You cannot reorder or change the default menu items in the top nav (ribbon).
  • Fields in red indicate that the field cannot be changed. In most cases this is due to legal requirements set by Adobe
  • This app does not allow you to change the name of web apps. To change the name of the web app, access the settings page for the web app.
  • You cannot place links to the site domain (a front end page of the same website the app is installed on) or an app domain.
    • New restriction from Adobe as of 04/09/2014 update
  • External links must be delivered securely, over the https protocol.
    • New restirction from Adobe as of 05/12/2014

Troubleshooting

Q: When I save my changes why do I not see the updates reflected in the admin menus?
A: If you are logged into the admin console using http you will need to refresh the browser before changes will appear.

Q: Why isn't my link loading when trying to load in the admin console?
A: Some companies prevent their website from being framed into another website. To see if this is the cause to your problem, open up the development console in your browser and see if a javascript error was triggered when trying to load the page. The error will say something along the lines of 'X-Frame-options' to 'SAMREORIGIN'.

Q: I don't like my changes, how do I put the menus back to how it was?
A: Click the Restore Default Settings button under the "Change Labels" tab.

Q: I can't find the "Admin Menu Labels" app anywhere in the menu
A: This can be caused by either the hiding of the menu link to the app using the app or there is an error in menu.json file created by the app. You can always access the app by going to it directly: http://YOURDOMAIN/Admin/AppLoader.aspx?client_id=solid-admin-menus. If there is an error in the file, going to the app URL directly should fix the menu.json file. If you are still seeing errors in Business Catalyst and your menus are not appearing, please contact support.
